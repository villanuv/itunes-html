Not iTunes 0.52 Release
New Notifications area. Main interface extended with more data. Added playlist count, upload times. Top bar with Playlist, Controls and Date dropdowns. More icons.

Longer titles now scroll, while shorter titles stay put. New status bar, with playlist count and duration. Added fullscreen. Enabled clearing Recently Played. Keyboard commands synced.

iTunes HTML
Originally a coding challenge from Belly in 2015. Used Rdio's service, and broke once they closed shop. Applied desktop UI to the web through HTML, CSS, JavaScript (AngularJS) and images.
